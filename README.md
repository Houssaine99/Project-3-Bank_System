# 🏦 Bank Management System (C++)

A console-based Bank Management System built using C++ and Object-Oriented Programming principles.
This project simulates real-world banking operations such as managing clients, transactions, and user authentication.

---

## 📌 Features

### 👤 Client Management

* Add new clients
* Update client information
* Delete clients
* Search for clients
* List all clients

### 💰 Transactions

* Deposit money
* Withdraw money
* Transfer between accounts

### 🔐 User Management

* Login system
* User permissions (if implemented)
* Secure access to system features

### 💱 Currency Exchange

* Currency conversion
* Exchange rate management

### 🖥️ Console UI

* Interactive menu system
* Multiple screens (Login, Main Menu, Transactions, etc.)
* Clean and structured user experience

---

## 🧠 Concepts Used

* Object-Oriented Programming (OOP)

  * Classes & Objects
  * Encapsulation
  * Separation of concerns
* File Handling (Data Persistence)
* Modular Design
* Console UI Design
